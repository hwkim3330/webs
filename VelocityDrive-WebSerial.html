Instructions

🔌 How to Connect:

  1. Connect your VelocityDRIVE board via USB cable
  2. Click "Connect" button above
  3. Select your device from the browser dialog
  4. Start using the Terminal or Commands tabs

📋 Requirements:

  • Chrome/Edge 89+ or Opera 75+
  • HTTPS connection (or localhost)
  • Microchip VelocityDRIVE board connected via USB

⚠️ Browser Support:

This application uses the WebSerial API which requires a Chromium-based browser. Firefox and Safari are not supported yet.

VelocityDRIVE WebSerial Terminal ================================ Connect your device first, then type commands below. Available commands: - device /dev/ttyACM0 type - device /dev/ttyACM0 get /ietf-system:system-state - device /dev/ttyACM0 get /ietf-interfaces:interfaces - device /dev/ttyACM0 yang Type 'clear' to clear this terminal.
